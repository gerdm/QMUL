Data-Set: Florentine families
Description: The network of marital relationship among Florentine
             families in the 15th century. Each node is a family, and
             a link between a pair of families indicates that one
             member of the first family had married one member of the
             other one. 
Author: S. Wasserman, K. Faust
Reference: S. Wasserman and K. Faust. Social network analysis: Methods
           and applications. Vol. 8. Cambridge university press, 1994.
Additional-Info: This data set is part of the accompanying material of the book
                 "Complex Networks: Principles, Methods and Applications", 
                 V. Latora, V. Nicosia, G. Russo, Cambridge University Press (2017)
                 ISBN: 9781107103184
                 If you use the data set, please add a reference to
                 the original paper and to the book above.
Book-Reference: Chapter 2, Figure 2.1
