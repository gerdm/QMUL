Data-Set: Elisa's Kindergarten network
Description: This data set contains the declared friendship relationships
             among a group of 3-5 years old children in a
             kindergarten.
Author: The data set was originally collected and curated by
        Vito Latora.
Additional-Info: This data set is part of the accompanying material of the book
                 "Complex Networks: Principles, Methods and Applications", 
                 V. Latora, V. Nicosia, G. Russo, Cambridge University Press (2017)
                 ISBN: 9781107103184
                 If you use the data set, please add a reference to the book above.
